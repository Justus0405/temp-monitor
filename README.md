# Temp-monitor

A CPU temperature monitoring script for Linux written in bash

## Dependencies

```plaintext
bash
```

## Installation

1. Clone the repository:

   ```shell
   git clone https://github.com/Justus0405/temp-monitor.git
   ```

2. Navigate to the directory:

   ```shell
   cd temp-monitor
   ```

3. Run the script:
   ```shell
   ./temp-monitor
   ```
